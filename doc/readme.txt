В самой курсовой работе:

1. Введение
Во введении нужно указать актуальность разработки web-приложений и популярность стека Java-технологий для разработки web-приложений. Также можно указать на такую особенность, как командная разработка, её преимущества и способы организации совместной работы.
В введении обязательно указывается цель и задачи работы.
Целью курсовой работы является разработка модуля ... в рамках командной разработки системы управления проектами.
Для достижения поставленной цели ставятся следующие задачи:
- описать функциональные требования для разрабатываемого модуля;
- с применением HTML и CSS создать прототипы страниц разрабатываемого модуля;
- спроектировать структуру базы данных и архитектуру приложения с применением шаблона проектирования ...;
- разработать SQL-скрипты по заполнению базы данных тестовыми данными для проверки функциональности разрабатываемого модуля;
- реализовать функционал модуля.

2. Анализ и постановка задачи
В данном разделе описать требования к модулю на основании задач с github.com (но в развёрнутом виде). Здесь же привести скриншоты прототипов соответствующих страниц.

3. Проектирование
Общее описание структуры базы данных с приведением схемы базы данных (ER-диаграммы) и листингов SQL-скриптов по созданию таблиц (03_create_tables.sql).
Общая характеристика архитектуры приложения и выбор средств реализации (описание web-приложения, СУБД MySQL, Java-технологий, библиотеки JDBC, Servlet API, JSP).
Теоретическое описание шаблона проектирования ... применительно к системе управления проектами.

4. Разработка
Описание SQL-скриптов по заполнению базы данных тестовыми данными (с листингами).
Описание разработанных классов (сущностей, DAO, сервисов, сервлетов, JSP-страниц) с приведением листингов.

5. Заключение
Подвести итоги решения задач, сформулированных во введении.
